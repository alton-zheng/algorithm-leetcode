## 代码

```java
class Solution {
    public int minCostToMoveChips(int[] position) {
        
        int firstP = 0, secondP = 0;

        for (int p: position) {
            if ((p & 1) == 0) {
                firstP++;
            } else {
                secondP++;
            }
        }

        return Math.min(firstP, secondP);
    }
}
```
