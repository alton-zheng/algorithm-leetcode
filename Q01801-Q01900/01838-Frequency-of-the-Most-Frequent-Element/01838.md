## 方法一： 排序 + 预和（preSum）
视频当中的方法如下： 
```java
class Solution {
    public int maxFrequency(int[] nums, int k) {

        // 首先需要排序
        Arrays.sort(nums);
        int res = 1;
        long preSum = 0;

        for (int left = 0, right = 0; right < nums.length; right++) {
            preSum += nums[right];
            while(nums[right] * (right - left + 1) - preSum > k) {
                preSum -= nums[left++];
            }
            res = Math.max(res, right - left + 1);
        }

        return res;
    }
}
```

## 方法二： 暴力破解
```java
class Solution {
    public int maxFrequency(int[] nums, int k) {

        Arrays.sort(nums);

        int maxFrequency = 0;

        int equalIdx = -1;

        for (int i = nums.length - 1; i >= 0; i = equalIdx == -1 ? i - 1 : equalIdx - 1) {
            int tmp = k;
            int curFrequency = 1;
            equalIdx = -1;
            for (int j = i - 1; j >= 0; j--) {

                if (nums[i] == nums[j]) {
                    curFrequency++;
                    equalIdx = j;
                } else {
                    tmp -= nums[i] - nums[j];
                    if (tmp >= 0) {
                        curFrequency++;
                    } else {
                        break;
                    }
                }
            }

            if (curFrequency > maxFrequency) {
                maxFrequency = curFrequency;
                if (maxFrequency == i + 1) {
                    break;
                }
            }
        }

        return maxFrequency;
    }
}
```
